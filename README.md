# Task 12: Log Monitoring & Analysis

## Objective
Detect security incidents by analyzing Linux and Windows authentication logs and understanding SIEM-based monitoring.

## Tools
- Linux logs (/var/log/auth.log)
- Windows Event Viewer
- Splunk Free (optional)

## Key Analysis
- Identified repeated failed login attempts
- Correlated failed and successful logins
- Detected suspicious authentication patterns

## Deliverables
- One-page PDF report
- Realistic log screenshots
- SIEM (Splunk) dashboard example

## Outcome
Hands-on incident detection and SOC analyst skills.
